# Task 4 - REST API with Flask

## Objective
Create a REST API that manages user data using Flask.

## Features
- GET all users
- GET single user
- POST new user
- PUT update user
- DELETE user
- Returns JSON responses
- Proper HTTP status codes

## Technologies Used
- Python
- Flask
- Postman / Curl

## How to Run
1. Install Flask:
   pip install flask

2. Run:
   python RestAPI.py

3. Open in browser or Postman:
   http://127.0.0.1:5000/users

## Key Concepts Used
- REST
- HTTP Methods
- JSON
- Flask Routing
- Status Codes
