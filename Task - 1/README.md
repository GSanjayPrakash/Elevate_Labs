# Task 1 - CLI Calculator App

## Objective
Build a command-line calculator supporting basic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous loop until user exits

## Technologies Used
- Python
- VS Code
- Terminal

## How to Run
1. Open terminal
2. Run: python calculator.py
3. Follow on-screen instructions

## Key Concepts Used
- Functions
- Loops
- Conditionals
- User Input
