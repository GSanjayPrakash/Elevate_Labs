# Task 2 - To-Do List CLI App

## Objective
Create a console-based to-do list manager with file storage.

## Features
- Add tasks
- View tasks
- Remove tasks
- Persistent storage using text file
- Continuous loop until user exits

## Technologies Used
- Python
- VS Code
- Terminal

## How to Run
1. Open terminal
2. Run: python todo.py
3. Follow menu options

## Key Concepts Used
- Lists
- File Handling
- String Manipulation
- Loops
- Conditionals
