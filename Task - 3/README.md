# Task 3 - News Headlines Web Scraper

## Objective
Scrape top headlines from a news website and store them in a text file.

## Features
- Sends HTTP GET request
- Parses HTML using BeautifulSoup
- Extracts headline tags
- Saves headlines into headlines.txt

## Technologies Used
- Python
- requests
- BeautifulSoup
- VS Code

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4

2. Run:
   python News_Scraper.py

3. Headlines will be saved in HeadLines.txt

## Key Concepts Used
- HTTP GET requests
- Web Scraping
- HTML Parsing
- File Handling
- Exception Handling
